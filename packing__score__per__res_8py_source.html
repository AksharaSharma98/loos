40  Compute packing score between each residue of a protein and a number of
41  small molecules in solution.

43  Summary of arguments
44  system_file: file describing the full system contents. For this program
45  to work correctly, it must contain connectivity information, so either a PSF or a PDB with CONECT records is best
46  protein_selection: selection string for the protein
47  probe_selection: selection string for the small molecules (will be

49  output_core: root name of the output files
50  --traj: one or more trajectory files
51 
52  Options
53  --skip value : number of frames to skip at the front of each trajectory
54  --stride value: how to step through the trajectory (default=1)
55  --no_h: remove hydrogens from the selection (recommended)
56  --pca: perform principal component analysis on the time series
57 
58  Packing score is a measure of packing between two groups or moieties
59  inspired by the Lennard-Jones function. It is the sum over all
60  inter-moiety pairs of atoms of 1/r^6. To my knowledge, this term
61  was first introduced in Grossfield et al, Proc. Nat. Acad. Sci.
62  USA, 2006, 103, 4888-4893.
63 
64  Output:
65  If output_core is set to "foo", then the following files will be
66  produced:
67 
68  foo_raw_score.dat: matrix of packing scores. Each row is the packing
69  score between 1 specific probe molecule against each
70  protein residue from 1 frame.
71  foo_ave.dat: column of per-residue summed packing scores
72 
73  If --pca is specified, principal component analysis is performed on the
74  matrix from foo_raw_score, producing 3 additional files:
75 
76  foo_var.dat: the fraction of the signal for each eigenmode
77  (essentially the eigenvalues)
78  foo_var_cum.dat: the cumulative variance, so you can assess how many
79  eigenmodes you should analyze)
80  foo_comp.dat: each column is an eigenvector,
81  representing an intensity for each residue. This is
82  useful for identifying residues that simultaneously
83  interact with an individual probe molecule, and thus are
84  candidate binding sites.
85 
86  Example command
87  packing_score_per_res.py system.psf 'segid == "PROT"' 'resname == "LIG"' score --no_h --pca --traj traj_1.dcd traj_2.dcd
88 
89  -- the protein is segid "PROT", while the ligand has residue name "LIG"
90  -- analyzes 2 trajectories (traj_1.dcd, traj_2.dcd)
91  -- automatically removes hydrogens from the selection

156 frame_index = 0

162 for t in args.traj[1:]:

165  skip=args.skip,

169 # pre-allocate storage as a numpy array

171 
172 frame_index = 0
173 for frame in vtraj:

175  for r in range(len(residues)):
176  for p in range(len(probes)):

178  scores[r, p, frame_index] += s
179  frame_index += 1
