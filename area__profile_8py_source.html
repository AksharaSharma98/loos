21 
22 system: system file (e.g. pdb, psf, parmtop, gro)
23 trajectory: trajectory file (periodic boundary information required)
24 skip, stride: how to move through the trajectory, skipping the first "skip" values and stepping by
25  "stride"
26 zmin, zmax: range of z-values to consider for area calculation
27 num_slices: number of slices to break the vertical span into
28 padding: floating point number specifying how many extra layers of atoms are generated
29  (see below)
30 all-selection-string: the set of atoms used to compute the voronoi decomposition
31 target-selection-string: set of atoms whose areas we report
32 
33 Notes
34  1) all selections are forced to be subsets of the initial selection. This is
35  necessary for the mapping of areas to work correctly.
36  2) this program assumes that the system has already been centered such that
37  the z location of the membrane isn't drifting (z-slices are absolute, not
38  relative to the membrane center) and such that the periodic box
39  is centered at x=y=0.
40 
41 Padding:
42  This program uses scipy Voronoi implementation, which is itself a wrapper around
43  Qhull. Qhull doesn't know anything about periodic boundary conditions as far as I
44  can tell, so we fake it by generating extra atoms so provide an environment around
45  the real ones; otherwise, you get insane areas for the atoms at the "edge" of the
46  box. You need to pick this value such that there are ~2-3 layers of atoms around
47  the real ones; in my experience, a value of 15 ang works well if you're selecting
48  all atoms or all heavy atoms. If your selection is more sparse (e.g. just phosphate
49  atoms), you will need to make it significantly larger, perhaps 30 ang). The best
50  test is to run with different pad values on a short trajectory and find the value
51  where the answer doesn't change. Since a larger pad value increases the number
52  of atoms used in the Voronoi decomposition, it does get more expensive, but it's
53  never too bad, and there's always a faster way to get a wrong answer.
54 
55 
56 
57 Example selection choice:
58  '\!hydrogen' 'segid == "RHOD"'
59  use all heavy atoms for the voronoi decomposition, then pick out the rhodopsin
60  molecule to get its area
61  (as a rule, you should get virtually identical answers with all atoms and all
62  heavy atoms, but the latter will be dramatically faster)
63
