8 """
9  native-hbs.py : Find the hydrogen bonds present in some reference
10  selection, then output their average occupancy over a trajectory
11  (and optionally a timeseries of those occupancies and a PDB of
12  the atoms in the HBs).
13 
14  The expected use-case is the commonly asked question, 'does this Hydrogen
15  bonded structure persist in the simulation?' The expectation is that the
16  user would have some molecular model (such as a PDB) where the H-bonded
17  structure is present (say, a Watson-Crick duplex) and a trajectory of that
18  molecule which may no longer retain the structure.
19 
20  You may optionally specify files to get not just the average HB occupancy in
21  your selection, but also the per-frame occupancy of each hydrogen bond in
22  the selection in an output file. You can also optionally provide an outfile
23  name for a PDB of the hydrogen bonded atoms the script finds. This makes for
24  easy visualisation: load that file and your ref up in a visualizer, then
25  view the model in sticks (or something smaller) and the H-bonded atoms as
26  spheres (or something bigger). Alternatively, you could have your structure
27  viewer draw HB for one 'molecule' then the other and see whether the script
28  found the ones you were hoping it would
29 
30  Example commandline with no optional arguments:
31  native-hbs.py -m xtal_molA.pdb -x traj_molA.dcd
32 
33  This will print the following values separated by spaces to stdout:
34  Number_frames Number_HBs Average_HBs
35 
36  Example commandline with all the optional arguments: native-hbs.py \
37  -m xtal_mol_A.pdb -x traj_mol_A.dcd -n native_hbs_found_A.pdb \
38  -o native_hbs_timeseries_A.dat -b 4 -a 20 \
39  -s '(resid < 4 || resid > 8) &&!backbone'
40 
41  Note that with all the options present the three values above are still
42  written to standard out, but the outfile specified by -o will contain the
43  occupancy of each found HB by frame so they can be readily recalculated from
44  those files if you chose not to redirect the output.
45 
46  Note also that the default selection string is 'all', which may cause the
47  program to crash and will produce confusing and unuseful output if your
48  model system has many waters present. For a PDB or NMR structure with no
49  ordered waters this is no issue but a model system containing a full solvent
50  box (such as a restart from a trajectory) or ordered waters will run into
51  this issue. The right way to handle this is to use the -s flag and select
52  the part of the model that is of interest.
53

133 # build list of donor and acceptor groups from putative groups.
134 for i in putative:
135  bound_to_hydrogen = False
136  for h in hydrogens:

138  donors.append((h,i)) # Associate donor with its hydrogen
139  bound_to_hydrogen = True

143 # list of donor-H-acceptor triples present in reference crds
144 ref_dhas = []
145 
146 # fill out the donor-H-acceptor list
147 for i in acceptors:
148  for j in donors:

150  ref_dhas.append(np.array([i, j[0], j[1]]))

153  # Note: remove must be applied last at this depth to avoid inconsistent list counters

166 # iterate over all the frames in the traj, checking each frame for each HB.
167 if args.outfile:

170  for frame in traj: # frame will be a new model that needs a new HBondDetector
171  frame_count += 1
172  total_inframe = 0
173  outform = [frame_count] # list that will aggregate a row of data in outfile
174  frame_hbs = loos.HBondDetector(args.bondlength, args.angle, frame)
175  for dha in ref_dhas:
176  if frame_hbs.hBonded(dha[0], dha[1], dha[2]): # if there's an HB, add to counters
177  total_hbs += 1 # add to total hbs
178  total_inframe += 1 # add to inframe total
179  outform.append(1) # add a 1 to the row and column you're on in outfile
180  else:
181  outform.append(0) # didn't find an HB so add a zero to row in outfile
182  outform.append(total_inframe) # add last column to outfile
183  outfile.write('\t'.join(map(str,outform)) + '\n') # write tab delim row to outfile
184 else: # same as above, but don't write outfiles since they weren't asked for if youre here
185  for frame in traj:
186  frame_count += 1
187  frame_hbs = loos.HBondDetector(args.bondlength, args.angle, frame)
188  for dha in ref_dhas:

190  total_hbs += 1 # if DHA observed in frame, add one to counter.
