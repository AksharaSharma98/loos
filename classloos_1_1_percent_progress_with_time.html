Detailed Description

Provide feedback by percent-complete with estimates of time remaining

This class provides basically the same functionality as the BasicProgress class, except that we override the update and finish functions to provide more detailed information. This class requires a counter that implements the timeRemaining() and fractionComplete() functions.
